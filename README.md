## 📦 How to Use

```bash
git clone https://github.com/arunardor/Dev-Charlie.git
cd Dev-Charlie
open index.html
``` 

## 💻 Sample Code Block

```html
<section id="projects">
  <h2>My Projects</h2>
  <div class="project">
    <h3>Portfolio Website</h3>
    <p>A personal portfolio site made with HTML, CSS, and JavaScript.</p>
  </div>
</section>
```
